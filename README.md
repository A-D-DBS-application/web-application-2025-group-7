## Extra information how to install/use the app 
    ## Installatie

    1. Zorg dat Python 3 geïnstalleerd is.

    2. Clone deze repository en ga in de projectmap:

        Typ in de terminal: cd web-application-2025-group-7/Flask
        
    3. (Aanbevolen) Maak en activeer een virtual environment:

        Aanmaken venv:
            Typ in de terminal: python -m venv venv

        Activeren venv:
            Windows:
                venv\Scripts\activate
            OF
            macOS / Linux:
                source venv/bin/activate
    4. Installeer de vereiste packages:

        Typ in de terminal: pip install -r requirements.txt
    
    ## Applicatie runnen

    1. Ga (indien nodig) naar de Flask-map:

        Typ in de terminal: cd web-application-2025-group-7/Flask
    
    2. Zorg dat de virtual environment geactiveerd is (zie boven).

    3. Start de Flask‑app:

        Typ in de terminal: python run.py

    4. Open in de browser:

        Ctrl+klik: http://127.0.0.1:5000

## Link to Kanban board (if used)

    Voor dit project hebben we geen apart online Kanban-board gebruikt. We hebben kort geëxperimenteerd met Miro, maar merkten al snel dat deze manier van werken niet goed aansloot bij ons kleine team van vier studenten.

    In plaats daarvan hebben we ons werk gecoördineerd via een gedeelde messengergroep. Na elke les over het project en na elke feedbacksessie met Gitoo maakten we samen een lijst met taken op korte termijn, die we onder elkaar verdeelden. Vragen, het delen van bestanden en voortgangsupdates verliepen allemaal via deze messengergroep.
